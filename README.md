<H1>Aplikacja Northwind</H1>
<H2>Stworzona za pomocą C# WPF, zgodnie z zaleceniami wykorzystano localdb SQLSERVER, z racji tego przed próbą kompilacji należy zaimportować bazę danych (załączyłem formę SQL)
